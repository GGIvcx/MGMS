# What does the Plugin do?
This plugin is designed to make the drops of gold when mined by a golden pickaxe make sense.\
# Supportet Versions
- Spigot 1.21.8

## Features
- Add drops to gold blocks mined by a golden pickaxe
- Supports Silktouch
- Supports Fortune
